# Тестовое задание: Laravel-разработчик

## 📝 Задача

Реализуйте веб-приложение на Laravel 10+, которое позволяет пользователям создавать короткие ссылки, отслеживать переходы и управлять своими ссылками через личный кабинет, если зарегистрировался.

## 📋 Функциональные требования

1. Аутентификация
* Пользователь должен иметь возможность зарегистрироваться и войти.

2. Создание ссылок
* Пользователь может указать оригинальный URL (например - https://example.com/page) и получить короткую ссылку (например - https://yourdomain.test/abc123).

3. Перенаправление
* При переходе по короткой ссылке должен происходить редирект на оригинальный URL.
* Переход должен быть зафиксирован в статистике.

4. Пользователь может:
* видеть список своих ссылок;
* удалять ссылку;
* просматривать статистику по каждой ссылке (см. ниже).

5. Статистика переходов. Для каждой ссылки сохраняется:
* IP-адрес пользователя;
* дата и время перехода;
* В кабинете пользователя можно посмотреть список переходов по каждой ссылке и общее количество кликов.

Большим плюсом будет реализация личного кабинета на базе админ-панели Filamentphp v3.
